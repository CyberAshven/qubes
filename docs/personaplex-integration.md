# PersonaPlex-7B Integration Plan (DEFERRED)

This plan has been deferred in favor of the streaming TTS approach (see `streaming-tts.md`).

PersonaPlex-7B is a full-duplex speech-to-speech model requiring 24GB+ NVIDIA GPU. While powerful, it's too hardware-intensive for most users. The streaming TTS approach achieves similar perceived responsiveness using existing providers on all hardware.

PersonaPlex may be revisited as a Tier 4 premium feature for power users with high-end NVIDIA GPUs.

See: https://huggingface.co/nvidia/personaplex-7b-v1
